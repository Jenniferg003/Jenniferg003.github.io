INFORMACIÓN GENERAL

HTML5 es la última versión de HyperText Markup Language (Lenguaje de Marcado de Hipertexto), que combina tres tecnologias web:
HTML, para construir la estructura de la página; CSS para mejorar la presentación; y JavaScript para agregar funcionalidad a los elementos HTML.
Sus etiquetas o marcas hacen que textos, imágenes y elementos multimedia se visualicen correctamente en la pantalla, y que las páginas web tengan un formato unificado,
estandarizado y sean compatibles con distintos navegadores como Chrome, Edge, Safari.
Por lo tanto en ésta pagina web se utilizan todas las etiquetas de htlm5, así como la implementación de su funcionalidad.

Pasos para crear una página web básica

  1. Crear un nuevo archivo con extensión .html.
  2. Escribir la estructura con las etiquetas básicas: !DOCTYPE, html, head, title, link, meta, body.
  3. Agregar un título a la página dentro de la etiqueta title.
  4. Incluir un contenido dentro del cuerpo de la página.
  5. Guardar el archivo y abrirlo en un navegador para ver el resultado.

Implementar buenas prácticas en el mundo de la programación

En el mundo de la programación, es fundamental aplicar buenas prácticas, y una de las más importantes es escribir código limpio. El código limpio prioriza la claridad, lo que facilita la lectura, la comprensión y la modificación del código. Escribir código legible reduce el tiempo necesario para comprender la funcionalidad del código, lo que conduce a tiempos de desarrollo más rápidos. Existe un libro que nos enseña sobre el tema Código Limpio.

"Los nombres de las variables, funciones y clases deben ser lo suficientemente descriptivos para que cualquier persona que lea el código pueda entender el propósito del código sin tener que leer el código mismo." Clean Code: A Handbook of Agile Software Craftsmanship por Robert C. Martin.

El Front-end y el Back-end

Front-end
El front-end es la parte de una página web o aplicación que los usuarios ven y con la que interactúan.

Algunas tecnologías comunes que se usan en el front-end son: HTML, CSS y JavaScript

Back-end
El back-end es la parte lógica de un sitio, este se encarga de la lógica de negocio, de recibir y devolver datos procesados a las apps y sitios web.

Algunas tecnologías comunes que se usan en el back-end son: Lenguajes de programación, bases de datos, Frameworks y servidores

El uso de etiquetas

Con el paso del tiempo y la transición desde versiones anteriores a HTML5 (por ejemplo, desde HTML4 o XHTML), hay muchas etiquetas HTML que han sido marcadas como obsoletas y se recomienda dejar de utilizarlas cuanto antes. Es muy posible que algunas incluso funcionen aún en navegadores actuales, pero esto ocurre por retrocompatibilidad y llegará un momento que dejarán de ser soportadas por el navegador.
